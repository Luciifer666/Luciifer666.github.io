COMMENT INSTALLER ?

Sur PC (Chrome ou Chromium) :
1. Télécharger l'extension :
   - Télécharge le dossier de l'extension (avec tous les fichiers nécessaires comme `popup.html`, `popup.js`, etc.) ou un fichier ZIP contenant l'extension.

2. Activer le mode développeur :
   - Ouvre Google Chrome.
   - Dans la barre d'adresse, tape `chrome://extensions` et appuie sur Entrée.
   - En haut à droite, active le mode développeur (un bouton situé juste en dessous du titre "Extensions").

3. Charger l'extension non répertoriée :
   - Clique sur Charger l'extension non empaquetée.
   - Dans la fenêtre qui s'ouvre, sélectionne le dossier de l'extension téléchargée (assure-toi qu'il contient le fichier `manifest.json`).
   - L'extension sera alors ajoutée à ton navigateur, et tu verras son icône apparaître dans la barre d'outils.

4. Utilisation :
   - Clique sur l'icône de l'extension pour l'utiliser.
   - Suis les instructions affichées à l'écran.

Sur PC (Firefox) :

1. Télécharger l'extension :
   - Télécharge le dossier de l'extension (avec tous les fichiers nécessaires comme `popup.html`, `popup.js`, etc.) ou un fichier ZIP contenant l'extension.

2. Activer le mode développeur :
   - Ouvre Mozilla Firefox.
   - Tape `about:debugging` dans la barre d'adresse et appuie sur Entrée.
   - Sur la page qui s'affiche, clique sur Ce Firefox (ou "This Firefox" si tu es en anglais).

3. Ajouter l'extension non répertoriée :
   - Clique sur Charger un module complémentaire temporaire.
   - Sélectionne le fichier manifest.json de ton extension. Cela ajoutera l'extension à Firefox pour la session en cours.

4. Utilisation :
   - L'extension sera installée et tu verras son icône dans la barre d'outils de Firefox.
   - Clique sur l'icône pour l'utiliser.

Remarque :
- L'extension restera active uniquement pour la session actuelle de Firefox. Si tu fermes et redémarres Firefox, l'extension devra être réinstallée de cette manière.

Sur Mobile (Android) :
Google Chrome sur Android ne permet pas d'installer directement des extensions comme sur PC. Toutefois, il existe une solution pour installer des extensions via un autre navigateur basé sur Chromium, comme Kiwi Browser ou Yandex Browser.

Avec Kiwi Browser :
1. Installer Kiwi Browser :
   - Télécharge Kiwi Browser depuis le Google Play Store : https://play.google.com/store/apps/details?id=com.kiwibrowser.browser.

2. Activer les extensions :
   - Ouvre Kiwi Browser et tape dans la barre d'adresse `chrome://extensions`.
   - Active le mode développeur en haut à droite.

3. Ajouter l'extension :
   - Télécharge ton fichier ZIP de l'extension.
   - Ouvre Kiwi Browser et retourne dans la page `chrome://extensions`.
   - Clique sur Charger l'extension non empaquetée et sélectionne le dossier décompressé de ton extension.
   
4. Utilisation :
   - L'extension devrait maintenant être installée. Tu peux l'utiliser en cliquant sur l'icône dans la barre d'outils de Kiwi Browser.

Avec Yandex Browser :
1. Installer Yandex Browser :
   - Télécharge Yandex Browser depuis le Google Play Store : https://play.google.com/store/apps/details?id=com.yandex.browser.

2. Suivre une procédure similaire à celle de Kiwi Browser pour ajouter l'extension via le mode développeur.

Remarque :
- Les extensions sur mobile ne sont pas aussi fluides que sur PC et peuvent présenter des limitations. Cependant, l'utilisation sur un navigateur mobile comme Kiwi ou Yandex permet d'en installer une grande majorité.
